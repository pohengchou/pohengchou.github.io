104Job Data Analysis

These projects demonstrate how I transformed the doubts and challenges I faced during my job search into actionable insights—by collecting, cleaning, and analyzing job market data through web scraping, ultimately guiding my own career decisions.

Step 1：104職缺資料爬蟲

使用 Python 中的 `requests` 以及 `pandas`，自動化爬取 104 職缺網頁，取得關鍵欄位如職稱、公司、薪資、技能需求等。

  • GitHub 連結

Step 2：匯入與 SQL 篩選預處理

將職缺資料匯入 Microsoft SQL Server，撰寫 SQL 指令進行欄位預篩選。

  • GitHub 連結

Step 3：資料清理

以資料視覺化工具（如 seaborn、matplotlib）分析薪資分布、技能需求排行、城市分布等，並嘗試提出可行的職涯建議。

  • GitHub 連結

Step 4：Power BI 視覺化探索

將清理後的 CSV 匯入 Power BI，製作互動式報表與圖卡，呈現技能熱度排行、地區職缺數、平均薪資趨勢等資訊，協助視覺化洞察市場趨勢。

  • 查看 Dashboard

Step 5：統計檢定分析

使用 SciPy 進行 t 檢定與卡方檢定，探索如「是否具備 Python 技能」對薪資的顯著性差異，與「不同產業」與「面議」是否具有統計關聯。

  • GitHub 連結

Step 6：簡報與專案總結

將整體專案成果彙整於 Canva 簡報中，包含數據來源、清洗流程、分析洞察與求職策略建議，使專案不僅具備技術性，亦能有效傳達價值與應用。
