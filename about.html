Poheng Chou(周柏亨)

Overview

我叫周柏亨，是一名具備雲端數據工程 (Cloud Data Engineering) 技能的 Data Engineer。 我專精於設計與維護高效能資料管線，並熟悉一系列現代數據棧核心技術，包括 Airflow、dbt、Docker、Terraform，以及 GCP (Google Cloud Platform) 雲端環境。

我畢業於清華大學教育心理與諮商學系，這段背景賦予我扎實的研究方法、統計分析與數據解讀基礎。這使我能從數據中，不僅看見數字，更能**理解其背後的人類行為與商業決策意涵**，為數據產品帶來更深入的洞察。

我將專注點轉向數據工程（Data Engineering），透過 **Udemy 上的深入課程** 和 **Data Engineer Zoomcamp 2025** 專案的實作訓練，我**系統化地掌握了核心技能**。

我的技術能力來自實戰經驗：我成功實現了完整的數據工程流程——從使用 Terraform 建立雲端基礎架構，到在 VM 上部署 Airflow + dbt 執行自動化 ELT pipeline，並最終在 BigQuery 中完成資料清洗、建模與星型模型建置。

我致力於在數據工程領域持續深耕，將所學應用於實際專案中，並積極拓展處理大規模數據 (Big Data) 的能力。我的近期目標是進一步掌握 Spark（分散式計算），結合 Airflow 的應用，成為一位能在高效能批次處理上發揮更大價值的全方位工程師。
